# HH Career Intelligence

AI-powered desktop SaaS dashboard для анализа вакансий с `hh.kz` и `hh.ru`, сравнения резюме с рынком и построения карьерных рекомендаций.

## Что умеет

- Обрабатывает ссылки вида `hh.kz/vacancy/123`, `almaty.hh.kz/vacancy/123`, `spb.hh.ru/vacancy/123` и ссылки с query string.
- Загружает списки ссылок из TXT, CSV и XLSX.
- Собирает вакансии напрямую из поиска HH во вкладке "Анализ рынка".
- Рассчитывает релевантность вакансий и исключает нерелевантные результаты из рыночной аналитики.
- Извлекает название, компанию, город, зарплату, полный текст, обязанности, требования и ключевые навыки.
- Ищет навыки по расширенному техническому словарю 500+ записей.
- Показывает детальную карточку вакансии внутри интерфейса.
- Строит ТОП-100 навыков, обязанностей и требований с процентом встречаемости.
- Сравнивает резюме DOCX, PDF и TXT с рынком вакансий.
- Экспортирует компактный набор отчетов: `Market_Report.xlsx`, `Resume_Gap_Analysis.xlsx`, `raw_data.json`.
- Собирается в EXE через PyInstaller.
- Новый интерфейс работает на PyQt6 в dark SaaS стиле.
- AI Advisor использует Gemini 2.5 Flash только на агрегированных данных.

## Установка

Требуется Python 3.12+.

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium
```

Для AI-функций задайте ключ:

```powershell
$env:GEMINI_API_KEY="your_api_key"
```

## Запуск

```powershell
python main.py
```

## EXE-сборка

```powershell
.\build.ps1
```

или двойной клик по `build.bat`.

После сборки файл появится здесь:

```text
dist/HH_Vacancy_Analyzer.exe
```

## Использование

### Анализ ссылок

Во вкладке "Вакансии" вставьте ссылку и нажмите "Анализировать ссылку" или загрузите файл TXT/CSV/XLSX. После загрузки отображаются счетчики:

- Всего вакансий
- Успешно обработано
- Ошибок

Двойной клик по строке открывает подробный просмотр: обязанности, требования, навыки, исходный текст, компания, зарплата и ссылка.

### Анализ рынка

Во вкладке "Анализ рынка" заполните:

- Поисковый запрос: `Системный администратор`, `DevOps`, `Linux`, `Technical Support`
- Регион: `Алматы`, `Казахстан`, `Москва`, `СПБ` или numeric area ID HH
- Количество страниц
- Минимальная релевантность: по умолчанию `50`
- Количество потоков: `3`, `5` или `10`

Приложение само соберет ссылки из поиска HH и обработает вакансии последовательно.
Для ускорения используется один Chromium и один BrowserContext на цикл анализа, несколько вкладок и кэш по ID вакансии.

### Релевантность

Для каждой вакансии рассчитывается `score`. В аналитику рынка попадают только вакансии с релевантностью не ниже заданного порога.

Положительные сигналы:

- название вакансии: системный администратор, sysadmin, support engineer, IT инженер, сетевой инженер;
- технические навыки: Linux, Windows Server, Active Directory, TCP/IP, DNS, DHCP, Mikrotik, Cisco, VMware, Hyper-V, Docker, Git, PowerShell, Bash, Zabbix, Grafana;
- текстовые маркеры: обслуживание серверов, администрирование серверов, техническая поддержка, сетевое оборудование, групповые политики, виртуализация, резервное копирование.

Отрицательные сигналы в названии сразу исключают вакансию: продавец, кассир, бариста, администратор кофейни, администратор клиники, администратор салона, офис-менеджер, бизнес-ассистент и похожие нерелевантные должности.

Во вкладке "Статистика" есть таблица "Исключенные" с причиной исключения. В экспорте создается отдельный `relevance_report.xlsx`.

### Статистика

Во вкладке "Статистика" доступны:

- ТОП-100 навыков
- ТОП-100 обязанностей
- ТОП-100 требований

### Резюме

Во вкладке "Резюме" загрузите DOCX, PDF или TXT. Приложение покажет:

- общий процент совпадения;
- навыки найденные в резюме;
- отсутствующие навыки;
- наиболее востребованные навыки рынка;
- рекомендации.

### Экспорт

Кнопка "Экспорт" создает отдельные файлы:

- `Market_Report.xlsx`
- `Resume_Gap_Analysis.xlsx`
- `raw_data.json`

`Market_Report.xlsx` содержит вкладки: "Вакансии", "Навыки", "Обязанности", "Требования", "ТОП навыков", "ТОП обязанностей", "ТОП требований", "Отфильтрованные вакансии".

## Ограничения

Приложение не использует API HeadHunter и работает только с публичной HTML-разметкой. Если HH изменит селекторы или покажет антибот-проверку, парсер может потребовать обновления.
